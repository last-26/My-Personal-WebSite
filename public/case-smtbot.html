03 · Stack

What the public surface can safely show.

Lean runtime, exchange-native IO, SQLite memory, read-only dashboard, and tests around the private core.

Data Plane
  • Bybit V5 WS closed-candle market data
  • KlineBuffer + cache rolling windows, reconnect recovery
  • MarketStateBuilder state from raw exchange rows
Risk & Portfolio
  • RR system standardized reward/risk math
  • Position sizing equity-aware exposure control
  • Circuit breakers loss, streak, drawdown, cooldown
Execution
  • pybit V5 client orders, positions, wallet, fills
  • Order router single controlled execution path
  • Position monitor exchange state reconciliation
Ops & Validation
  • aiosqlite journal audit trail for runtime events
  • FastAPI + uvicorn read-only terminal dashboard
  • pytest 63 files, roughly 975 cases
